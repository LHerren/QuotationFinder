QuotationFinder
   
 

QuotationFinder allows users of the TLG CD-ROM (Thesaurus Linguae Graecae) or the CLCLT CD-ROM (Library of Latin Texts) to find quotations and allusions in these digital libraries.

QuotationFinder searches for quotations of any text you like. The search is performed on files you export from the CD-ROMs. What you get is an ordered list of all quotations and allusions found, with the exact quotations first and the loose verbal parallels last.

   
 
Requirements
  • You need to have your own access to the TLG and/or CLCLT CD-ROMs.
    TLG: You need to be a user of Pandora for Macintosh or TLG Workplace for Windows—let me know if you would like to use QuotationFinder with other TLG software. Macintosh users, please be aware that Pandora will not run on Intel-based Macs or Macs with OS X 10.5 or higher—instead, you might consider using TLG Workplace by running Windows on your Mac.
    CLCLT: You need to have your own access to the CD-ROMs, either directly or via network.
  • Windows:
    If you have Windows XP, you are ready to go.
    If your operating system is Windows 95 or later, but older than Windows XP, and you want to search TLG files, download and install a Unicode polytonic Greek font such as TITUS Cyberbit Basic, free for non-commercial use.
    If your version of Windows is older than Windows 95, you will only be able to use QuotationFinder with the Latin CLCLT.
    Make sure you are using a fairly recent Web browser (Internet Explorer 4 or later, Netscape 4 or later, Opera 6, Mozilla, Firefox—for details click here.)
  • Macintosh:
    If you have Mac OS X 10.2 or later, you are ready to go.
    If you have Mac OS X, but your version is older than 10.2 and you want to search TLG files, download and install a Unicode polytonic Greek font such as TITUS Cyberbit Basic, free for non-commercial use.
    If your version of the Mac OS is older than Mac OS X, you will only be able to use QuotationFinder with the Latin CLCLT.
    Make sure you are using a fairly recent Web browser (Google Chrome, Safari, OmniWeb, Opera, Mozilla, Firefox.)
